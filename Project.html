Le HTML:

Le HTML (HyperText Markup Language) est un langage de balisage utilisé pour créer et structurer le contenu des pages Web.

Le CSS:

Le CSS (Cascading Style Sheets) est un langage de feuilles de style utilisé pour définir la présentation et la mise en forme du contenu HTML sur une page Web.

Le Java Script:

JavaScript est un langage de programmation de haut niveau utilisé principalement pour rendre les pages Web interactives et dynamiques. Il permet d'ajouter des fonctionnalités, de la logique, et de manipuler le contenu d'une page Web. JavaScript est exécuté côté client, ce qui signifie que le code s'exécute dans le navigateur Web de l'utilisateur, offrant ainsi une expérience utilisateur interactive.
